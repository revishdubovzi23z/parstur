# Antigravity Tracker

Персональный менеджер медиа-каталога и синхронизатор релизов.

## Что это такое?

Antigravity Tracker — это инструмент для тех, кто хочет следить за новыми релизами фильмов и сериалов, не полагаясь на сторонние сервисы с рекламой. Он собирает данные с Rutor, HDRezka и дополняет их метаданными из Kinopoisk и TMDB.

### Основные возможности:
*   **Синхронизация с Rutor**: Автоматический поиск новых раздач по заданным правилам (год, качество, категория).
*   **Интеграция с HDRezka**: Поиск прямых ссылок на стриминг, просмотр серий и сезонов прямо в интерфейсе.
*   **Метаданные**: Автоматическое подтягивание постеров, описаний и рейтингов (КП/IMDb).
*   **Коллекции**: Удобная группировка контента по папкам («Хочу посмотреть», «Избранное» и т.д.).
*   **Умный поиск**: Полнотекстовый поиск по базе (FTS5) и фильтрация по годам/рейтингам.
*   **Аудит**: Логирование изменений и возможность отката привязок к метаданным.

## Быстрый старт

### 1. Через Docker (Рекомендуется)

Самый простой способ запустить проект — использовать Docker Compose.

1.  Клонируйте репозиторий.
2.  Создайте файл `.env` на основе `.env.example` и заполните API-ключи.
3.  Запустите контейнеры:
    ```bash
    docker-compose up -d
    ```
4.  Приложение будет доступно по адресу `http://localhost:8000`.

### 2. Локальный запуск (Python 3.10+)

1.  Создайте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate  # или venv\Scripts\activate на Windows
    ```
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3.  Запустите сервер:
    ```bash
    python main.py
    ```

## Настройка

Все настройки задаются через переменные окружения или файл `.env`. Подробное описание каждой переменной можно найти в [`.env.example`](.env.example).

Ключевые переменные:
*   `AUTH_USER` / `AUTH_PASS_HASH`: Логин и хеш пароля для входа.
*   `TMDB_API_KEY`: Для получения описаний и постеров (рекомендуется).
*   `REZKA_EMAIL` / `REZKA_PASSWORD`: Для доступа к личным коллекциям и синхронизации статусов на HDRezka.
